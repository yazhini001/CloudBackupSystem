# ☁️ Cloud-Based Data Backup System

A Cloud-Based Data Backup System developed using **Flask**, **MySQL**, and **Cloudinary**. This application allows users to securely upload, organize, search, download, and manage files through a simple web interface.

---

## 📌 Project Overview

The Cloud-Based Data Backup System is a web application designed to provide secure cloud storage for users. It enables authenticated users to upload files to Cloudinary while maintaining file information in a MySQL database.

The project was developed as part of an internship to demonstrate cloud storage integration, database management, and secure user authentication using Flask.

---

* ✅ User Registration

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CloudBackupSystem.git
```

2. Open the project folder

```bash
cd CloudBackupSystem
```

3. Install required packages

```bash
pip install -r requirements.txt
```

4. Configure your MySQL database.

5. Configure your Cloudinary credentials.

6. Run the application

```bash
python app.py
```
